Welcome to Flask-Zipper's documentation!
========================================

Pythonic response compressor for response of Flask

Flask를 위한 view decorator 기반의 response 압축 라이브러리. Flask Large Application Example에서 직접 구현해 사용하
던 몇 가지 view decorator에서 출발했고, MDN의 표준 네가지 중 자주 사용되는 세 가지인 `br`, `deflate`, `gzip` 압축방식을
지원합니다.
